CloudByte ElastiStor plug-ins

The CloudByte ElastiStor plug-ins provide integration and interoperation with virtualization environments, cloudstacks, and applications.

The following plug-ins are available with ElastiStor:

  • CloudByte Virtualization Service plug-in for VMware VCenter
  • CloudByte ElastiStor plug-in for Apache CloudStack
  • CloudByte ElastiStor plug-in for OpenStack
  • CloudByte ElastiStor Recovery Manager for SQL Server plug- in

With OpenStack and CloudStack integration, ElastiStor is the perfect fit for SAN cloud storage. The RMSQL plug-in provides MSSQL application consistent snapshots.
