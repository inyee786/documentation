LSI SAS switch configuration for ElastiStor

This document explains how to manually configure a Serial Attached SCSI (SAS) switch. Also, the document explains how to use the SAS Domain Manager to create storage configurations in the SAS domain.

Log in to the SAS switch

The following table provides the default values that you have to specify when you log in to the LSI SAS Domain Manager.

Option Value
IP address 192.168.1.100
Username/Password admin/admin
Activation zone password lynx

 

Configure the SAS switch

  1. In a browser of your choice, specify the following URL: http://192.168.1.100

    Note: If you have already configured your SAS switch, specify the new IP address instead of the default one.

  2. In the management GUI, click Devices.

    3. Note:All the devices connected to the SAS switch are listed. If the devices are connected in a daisy chain fashion, the devices are listed in a tree structure.

  3. Create Aliases.
    1. Click the SAS device in the tree (in the left pane) for which you want to create an alias.
    2. Specify an alias of up to 24 characters in the Enter alias field.
    3. Click Apply.

  4. Create Zone Groups.

    5. Note:  CloudByte recommends you to create a dedicated Zone Group for each initiator/target connected to the SAS switch.

    1. Click Create Zone Group in the left pane.
    2. Specify a name in the Zone Group Name field.
    3. Select members for the Zone Group from the drop-down list.
  5. Create Zone Set to map the zone groups which need to communicate with each other.
    The following screen shows JBOD104106GRP configured to communicate with
    • Node N104GRP
    • Node N106GRP
  6. Activate zone set for the configurations to take effect.
    1. Click Activate/Deactive Zone Set in the left pane.
    2. Select Activate a Zone Set option.
    3. Select a Zone Set from the drop-down list.
    4. Specify the default zone manager password: lynx
