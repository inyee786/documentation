Configuring hardware RAID

This section explains how to configure hardware RAID and create virtual disk in scenarios where you use Dell PowerEdge R620/R720/R820.

  • Switch on your Dell PowerEdge R620/R720/R820 system.
  • Press <F2> to enter System Setup in the BIOS startup screen.

  • Click Device Settings from the System Setup Main Menu.

  • Click the RAID Controller link in Device Settings.

  • To view the properties of a physical disk:
    • Click Physical Disk Management from the Configuration options.
    • (In the following screen) Click View Physical Properties.

  • To create a virtual disk by selecting the RAID level:
    • Click Virtual Disk Management from the Configuration Options.
    • (In the following screen) Click Create Virtual Disk

  • In the following screen, configure the virtual disk parameters and then click Apply Changes.

  • Select RAID1 and then click Select Physical Disks. Configure the settings as shown in the following screen and then click Apply Changes.

  • Click Yes to confirm.
  • Check if the created virtual disk is marked as a bootable device.

  • Click Back and then follow the procedures to complete the wizard.
