Backing up interface

Prerequsite

The NIC and the backup NIC should be on the same network.

Procedures

  • In the Nodes page (Elasticenter > Navigation pane > Node), click Network Interface Cards (Actions icon > Network Interface Cards)
  • Click Configure Backup Interface.
  • In the dialog, select a NIC as the backup from the drop-down list.
  • Click OK

Back up LAGG

For the LAGG that you have added (Add LAGG) you can set up a back up. The following sections explain the procedures. Backing up your lagg assures that the VSMs are always accessible.

Assumptions

All the NICs in the primary LAGG and backup LAGG should be on the same network.

Procedures

  • In the HA Groups page (Elasticenter > Navigation pane > HA Group), click Manage LAGG interface (Actions Icon > Manage LAGG interfaces).
  • In the Actions icon, click Configure Backup Interface.
  • Specify the following details:
    • For LAGG interface 1, select the LAGG for which you want to configure back up.
    • In the LAGG interface 2 field, select the LAGG for back up.
  • Click OK
