Installation options

The installation wizard provides the following installation options.

  • ElastiCenter: Installs only ElastiCenter, which is the central administrative console to manage your Node. Choose if you prefer to install Node later. For procedures, see Installing CloudByte ElastiCenter.
  • Node: Installs only the Node, which is managed using ElastiCenter. Choose if you prefer to install ElastiCenter later. For procedures, see Installing the Storage Node.
  • Both ElastiCenter and Node: Installs both ElastiCenter and Node. This installation is not recommended for production deployments. For procedures, see Installing CloudByte ElastiCenter.
