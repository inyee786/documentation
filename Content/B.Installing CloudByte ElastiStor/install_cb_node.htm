Installing Node

  1. Insert the CloudByte ElastiStor installation disk into your computer's DVD or CD drive and boot up your Node Server (ensure that the disk drive is set as your system's primary boot device). The installation starts up the boot options appear.By default, the boot option Boot CloudByte ElastiStor is selected. So you need not interfere.
  2. The welcome screen appears and then in the following screen, accept the license information
  3. Select the installation option. For details, see "Installation options"

4. At prompt, Specify the ElastiCenter IP address and then click OK.

5. Follow the procedures from steps 4 in the section "Installing ElastiCenter" to complete the Node installation process
