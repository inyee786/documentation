Configuring IPMI settings

Set up IPMI configuration

  1. Switch on one of the Nodes in your CloudByte ElastiStor Appliance.
  2. Press DEL to enter the Supermicro BIOS setup.
  3. In the following screen, use the right arrow key and navigate to IPMI tab.
  4. Use the down arrow key and navigate to BMC Network Configuration and then press enter.
  5. Press enter at Update IPMI LAN Configuration option and set it to YES.
  6. At the prompt, select Static (press enter) as the Configuration Address Source.
  7. Press enter and specify the Station IP address (IP address of the IPMI management console).
  8. Press enter and specify the Subnet Mask (the standard IPv4 subnet mask, for example 255.255.255.0).
  9. Press enter and specify the Gateway IP address (the IPv4 gateway IP address, for example 10.0.0.1).
  10. Press F4 to save the configurations and exit the BIOS setup.
  11. To access ElastiCenter,
    • Launch Mozilla Firefox Web browser.
    • Specify the URL in the following format: https://ManagementConsole_IP_Address/page/login.html
  12. Specify the following default credentials:

    Username admin
    Password password
